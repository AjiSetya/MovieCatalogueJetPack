# Movie Catalogue - Dicoding JetPack

Pantau katalog film atau serial TV, dan simpan pilihan favorit kamu ❤️

Project ini menggunakan :
- Kotlin
- Constraint Layout
- Shared Element Transition
- Retrofit
- View Model
- View Binding
- Data Binding
- MVVM
- Paging
- Room
- Unit Test
- Instrument Test
